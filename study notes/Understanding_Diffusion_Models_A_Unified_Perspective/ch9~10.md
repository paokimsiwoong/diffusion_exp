# [*Score-based Generative Models*](../../README.md#table-of-contents)
---
---

> Score-based 모델은 이번 코드 구현에 다루지 않으므로 일단 생략

**Equation 152:**

```math
p_{\boldsymbol{\theta}}(x) = \frac{1}{Z_{\boldsymbol{\theta}}}e^{-f_{\boldsymbol{\theta}}(x)}
```
<br>

> $Z_{\boldsymbol{\theta}}$는 $\int p_{\boldsymbol{\theta}}(x) dx = 1$을 만족시키기 위한 normalizing constant

**Equation 153~156:**

```math
\begin{aligned}
\nabla_{x} \log p_{\boldsymbol{\theta}}(x)
&= \nabla_{x}\log\left(\frac{1}{Z_{\boldsymbol{\theta}}}e^{-f_{\boldsymbol{\theta}}(x)}\right)\\
&= \nabla_{x}\log\frac{1}{Z_{\boldsymbol{\theta}}} + \nabla_{x}\log e^{-f_{\boldsymbol{\theta}}(x)}\\
&= -\nabla_{x} f_{\boldsymbol{\theta}}(x)\\
&\approx \mathbf{s}_{\boldsymbol{\theta}}(x)
\end{aligned}
```
<br>

**Equation 157:**

```math
\mathbb{E}_{p(x)}\left[\left\lVert \mathbf{s}_{\boldsymbol{\theta}}(x) - \nabla\log p(x)\right\rVert_2^2\right]
```
<br>

**Equation 158:**

```math
x_{i+1} \leftarrow x_i + c\nabla\log p(x_i) + \sqrt{2c}\boldsymbol{\epsilon},\quad i = 0, 1, ..., K
```
<br>

**Equation 159:**

```math
p(x) = c_1p_1(x) + c_2p_2(x)
```
<br>

**Equation 160:**

```math
p_{\sigma_t}(x_t) = \int p(x)\mathcal{N}(x_t; x, \sigma_t^2\mathbf{I})dx
```
<br>

**Equation 161:**

```math
\arg\min_{\boldsymbol{\theta}} \sum_{t=1}^T\lambda(t)\mathbb{E}_{p_{\sigma_t}(x_t)}\left[\left\lVert \mathbf{s}_{\boldsymbol{\theta}}(x, t) - \nabla\log p_{\sigma_t}(x_t)\right\rVert_2^2\right]
```

# [*Guidance*](../../README.md#table-of-contents)
---
---

> 조건부 이미지 생성 $p(x | y)$은 이번 구현에서 다루지 않으므로 일단 생략

```math
p(x_{0:T}) = p(x_T)\prod_{t=1}^Tp_{\boldsymbol{\theta}}(x_{t-1}|x_t)
```
<br>

**Equation 162:**

```math
\begin{aligned}
p(x_{0:T}|y) = p(x_T)\prod_{t=1}^Tp_{\boldsymbol{\theta}}(x_{t-1}|x_t, y)
\end{aligned}
```
<br>

```math
\hat{x}_{\boldsymbol{\theta}}(x_t, t, y) \approx x_0, \quad \hat{\boldsymbol{\epsilon}}_{\boldsymbol{\theta}}(x_t, t, y) \approx \boldsymbol{\epsilon}_0, \quad \text{or} \quad \mathbf{s}_{\boldsymbol{\theta}}(x_t, t, y) \approx \nabla\log p(x_t|y)
```
<br>

**Equation 163~165:**

```math
\begin{aligned}
\nabla\log p(x_t|y) &= \nabla\log \left( \frac{p(x_t)p(y|x_t)}{p(y)} \right)\\
&= \nabla\log p(x_t) + \nabla\log p(y|x_t) - \nabla\log p(y)\\
&= \underbrace{\nabla\log p(x_t)}_\text{unconditional score} + \underbrace{\nabla\log p(y|x_t)}_\text{adversarial gradient}
\end{aligned}
```
<br>

**Equation 166:**

```math
\begin{aligned}
\nabla\log p(x_t|y) &= \nabla\log p(x_t) + \gamma\nabla\log p(y|x_t)
\end{aligned}
```
<br>

**Equation 167:**

```math
\begin{aligned}
\nabla\log p(y|x_t) = \nabla\log p(x_t|y) - \nabla\log p(x_t)
\end{aligned}
```
<br>

**Equation 168~170:**

```math
\begin{aligned}
\nabla\log p(x_t|y)
&= \nabla\log p(x_t) + \gamma\left(\nabla\log p(x_t|y) - \nabla\log p(x_t)\right)\\
&= \nabla\log p(x_t) + \gamma\nabla\log p(x_t|y) - \gamma\nabla\log p(x_t)\\
&= \underbrace{\gamma\nabla\log p(x_t|y)}_\text{conditional score} + \underbrace{(1 - \gamma)\nabla\log p(x_t)}_\text{unconditional score}
\end{aligned}
```
